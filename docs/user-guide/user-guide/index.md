# Guia do Usuario — SmartReader Gateway Web UI

Manual de operacao da interface web do SmartReader Identix Gateway.

## Publico-alvo

Operadores e tecnicos responsaveis por configurar, monitorar e gerenciar leitores RFID Identix EZ700 atraves da interface web do gateway.

## Pre-requisitos

- Navegador web moderno (Chrome, Edge, Firefox ou Safari)
- Acesso de rede ao gateway (porta padrao: 8080)
- Credenciais de acesso (se autenticacao estiver habilitada)
